## SD Interface Creditsafe Releases

### 2.0.0

#### Enhancements

- The Licence Activation and Setup Wizard was added to the app.

- Permission sets were created for SD Interface Creditsafe.

- The Company Registration Number was added to the Search Criteria on the SD Interface Creditsafe Company Search page.

- The SD Interface Creditsafe Report FactBox was surfaced on the Sales Order page and the Purchase Order page.

- A freeze frame was added to the SD Interface Creditsafe Credit Reports list after the Name column.

- A KPIs FastTab was added to the Setup Card.

- A new action was added to the Setup Card called View Our Apps. This action opens a page on AppSource pointing to all our Simply Dynamics Ltd apps.
